## Introducción

Proyecto de Fundamentos del Hardware | 1º ASIR

Bienvenid@ al repositorio del proyecto sobre Seguridad en el Trabajo aplicada al entorno informático. Este trabajo analiza los factores de riesgo, las medidas preventivas y la normativa vigente para profesionales del sector IT.

Para realizar este trabajo, cada integrante del grupo ha investigado y desarrollado una parte del contenido. Posteriormente, se ha organizado toda la información en conjunto para darle una estructura clara dentro del repositorio.
